Using the given patient information/HPI, create a handout with a comprehensive evidence-based plan for treatment. Start with a section on "Most Important Points". Then add sections regarding "Lifestyle" for diet/exercise/sleep/psych, "Non-pharmacological Interventions", "Medication Considerations", "Lab and Imaging" with recommendations for monitoring. Do not include a section if there are no interventions recommended for that section. This handout should use patient-friendly language. Provide as much information as necessary and do not focus on brevity.
