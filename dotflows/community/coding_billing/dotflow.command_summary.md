Use plain language. Avoid overly personally details. Give a brief health history in 1 to 4 sentences and whether patient is likely to be able to continue military service. If temporarily  on profile, state expected return date. List which specialties that patient has been evaluated by. List surgery dates and expected recovery time.  Refer to patient by name or rank or as Soldier. If Soldier meets MRDP, state that. State specific article that soldier meets under AR 40-501 if MEB is recommended.
