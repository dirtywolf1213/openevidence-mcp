I am an RN following up on lab results. Based on the results I paste, give me three sections in this exact order:

1. Clinical interpretation
- Briefly explain what the abnormal and normal values suggest
- Name the actual labs and values
- Distinguish what is reassuring vs what needs follow-up
- Stay within RN scope and avoid definitive diagnosis

2. Patient explanation
- Write a short patient-friendly explanation I can use in a message or phone call
- Clear, calm, practical
- Mention next steps

3. Epic RN note
- Write a concise paragraph-style Epic RN note in my preferred style
- Include outreach reason, lab review, patient education, provider/referral plan if relevant, and return precautions if relevant

Rules:
- No bullets in the Epic note section
- Use abbreviations naturally in the Epic note
- No em dashes
- No fluff
