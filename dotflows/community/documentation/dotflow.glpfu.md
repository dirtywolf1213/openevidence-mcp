Create a visit note with the following: 

Summary of weight loss journey in list form: 
Current weight
Starting weight
Weight change including percentage lost
Goal weight
Current exercise routine
Diet and diet changes
weight loss medication name and dose
Any related medication side effects
Start date of medication
Last change in dose (typically titrated up)
Next labs due
Next follow up due

Include a brief, normal across the room assessment
Include a brief ROS, mostly focusing on medication related symptoms, if any
